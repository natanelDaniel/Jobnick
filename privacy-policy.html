Privacy Policy - LinkedIn Job Manager Extension

Extension Overview

The LinkedIn Job Manager Extension is a Chrome extension designed to automate and streamline the LinkedIn job application process. It uses AI assistance to help users apply to jobs more efficiently.

Data Collection

This extension collects and processes the following types of user data:

Personally Identifiable Information

  • Name, address, email address, and other profile details
  • Job preferences and career information
  • Resume and cover letter content

Purpose: To pre-fill job application forms and streamline the application process.

Web History

  • LinkedIn page URLs and titles
  • Navigation patterns within LinkedIn domains

Purpose: To detect job listing pages and application forms for automation.

User Activity

  • Form interactions and clicks
  • Text input for form filling
  • Page navigation and scrolling

Purpose: To automate form filling and application submission processes.

Website Content

  • Job descriptions and requirements
  • Form field labels and structure
  • Page text content for AI analysis

Purpose: To extract job requirements and provide intelligent form filling suggestions.

Data Storage and Processing

  • Local Storage: All user data is stored locally in the user's browser
  • No Server Storage: We do not maintain any servers to store user data
  • AI Processing: Job-related content is sent to Google Gemini AI API for analysis and personalized responses

Data Usage

All collected data is used exclusively for:

  • Automating LinkedIn job applications
  • Pre-filling application forms
  • Generating personalized application responses
  • Improving the user experience

Data Sharing and Third Parties

We do not sell, rent, or trade user data.

The only third-party service we use is Google Gemini AI API, which is essential for:

  • Analyzing job descriptions
  • Generating personalized application responses
  • Providing intelligent form filling suggestions

Data sent to Google Gemini AI is processed according to Google's privacy policy and is not stored permanently.

Data Security

  • All data is stored locally in the user's browser
  • No data is transmitted to our servers
  • Data is only sent to Google Gemini AI when actively processing job applications
  • Users can clear all stored data by uninstalling the extension

User Rights

Users have the right to:

  • Access all data stored by the extension
  • Delete all stored data by uninstalling the extension
  • Control what data is collected through extension settings
  • Opt out of AI processing features

Updates to This Policy

This privacy policy may be updated from time to time. Users will be notified of any significant changes through the extension or this webpage.

Contact Information

For questions about this privacy policy or data handling practices, please contact us through the GitHub repository or extension support channels.

Compliance

This extension complies with:

  • Chrome Web Store policies
  • General Data Protection Regulation (GDPR) principles
  • California Consumer Privacy Act (CCPA) requirements
